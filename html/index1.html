About

The design of Raspbino is proposed for home safety. This system Automates with the help of sensor and alerts the owner about the home security and updates on smart gadgets by VOIP automated call and as an emergency measure the system will Locks home and also turnoff the power supply, while activating the alarm. The additional advantage of the system is that it continuously monitors the level using smart sensor. The device ensures safety and prevents suffocation and explosion due to gas leakage or electric malfunctioning. This project is implemented using raspberry pi which eases communication between owner and the internet protocol assigned system remotely from any network reachable area. This system also provides some additional interactive features like saying good morning and good night to the owner and notifying the owner with the basic precautions whenever it is necessary.

Keywords: LPG gas cylinder monitoring, VOIP call, remote access, gas leakage detection and prevention, GSM, Raspberry pi, Alarm.
